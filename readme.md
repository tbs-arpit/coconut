## How to use it?

Inside document ready you can do this.

```
$('#coconut').coconut({
    thickness: 1.15,
    values: [{
        amount: 10,
        color: "#001f3f"
    },
    {
        amount: 30,
        color: "#b58484"
    }],
    centerIconSrc: "//secure.fbstatic.com/images/Empowr/blue-icons/icon-thumbs-up.svg",
    centerValue: 5
});
```
